# Guida Piano City Milano 2026

Mini-guida statica per scegliere i concerti di Piano City Milano 2026.

## Cosa contiene

- 266 concerti dal programma ufficiale.
- Filtri per giorno, priorita, genere, tipologia, fascia oraria, GAM/Piano Center, prenotazione, salvati e location multiple.
- Link a scheda ufficiale, prenotazione quando disponibile, Maps, YouTube e Spotify.
- Shortlist degli eventi imperdibili e planner locale con salvataggio nel browser.

## Pubblicazione

Il sito e' statico: basta aprire `index.html` oppure pubblicare la cartella su GitHub Pages.
